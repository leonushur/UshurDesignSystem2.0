# Codebase Search

Delegate to the codebase-explorer for fast search:

```
@codebase-explorer Find $ARGUMENTS in the codebase. Provide file locations, code snippets, and related patterns.
```

Explorer can:
- Find components by name
- Search for usage patterns
- Locate design token usage
- List stories for components
- Check exports and dependencies
