# Design System Audit

Delegate to the design-system-auditor to check for:
- Token compliance (no hardcoded colors)
- Typography consistency
- Spacing system adherence
- Accessibility standards
- Component architecture

```
@design-system-auditor Audit $ARGUMENTS for design system compliance, checking token usage, accessibility, and code quality.
```
